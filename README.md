# Cerulean
Cerulean AI

Learning Without End

Cerulean is my long-term Python project, it’s an AI assistant designed mainly for students or people with big ambitions, who want to turn their long-term goals into reality.

Mission

To help people like students, startup founders, professionals, and anyone who wants to learn achieve goals that seem impossible. Unlike many other AI assistants, Cerulean is a voice-first AI companion with the ability to see your screen (this can be enabled or disabled) that understands what you're working on and helps you make progress toward long-term goals.

Vision

We aim to provide:
Goal planning, 
Learning roadmaps, 
Project guidance, 
Long-term memory, 
Tools for growth, 
Voice input, 
Screen sharing features
